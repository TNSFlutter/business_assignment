# Assignment_March_2024

This Flutter project is a sample application that demonstrates fetching business data from an API and displaying it in a Flutter app.

## Getting Started

To get started with this Flutter application, follow the steps below:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/TNSFlutter/business_assignment.git


2. **Navigate to the Project Directory:**
   ```bash
     cd business_assignment
   
3. **Run the App:**
   ```bash
     flutter run
This will launch the app on your connected device or emulator.

## Explore the App:

->The app starts with a splash screen that lasts for 3 seconds.
->After the splash screen, you will be directed to the login screen.
->Click on the "Login" button to access the business screen.
->The business screen displays a list of businesses fetched from an API.
->Tap on any business item to navigate to the detail screen and view more information.


## Project Structure
The project follows a standard Flutter project structure:
